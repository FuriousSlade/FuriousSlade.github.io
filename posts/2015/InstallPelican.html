Install pelican

安装

通过pip安装：

pip install pelican

通过git安装：

git clone git://github.com/getpelican/pelican.git cd pelican python setup.py install

建立项目

mkdir myblog
cd myblog
pelican-quickstart

回答一系列问题后项目建立。

目录结构：

Makefile            # Makefile
content             # 页面和文章静态文件放这个目录
develop_server.sh   # 开启本地测试web服务器,一般不使用。
output              # 站点生成目录
pelicanconf.py      # 配置文件
publishconf.py      # 配置文件

安装Cygwin

在开始写博客前，如果使用的是windows操作系统，需要先安装Cygwin来解决使用make命令的问题。 下载后安装Cygwin，一路下一步。到select packages页面时如图选择。
